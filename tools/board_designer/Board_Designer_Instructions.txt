Board Designer - Quick Setup

1) Extract this ZIP into your X-in-Line project folder.

2) Run the designer with Python:
   python board_designer.py

3) In the designer:
   - Set width/height if needed.
   - Draw obstacles with click/drag.
   - Click "Generate Code".

4) Copy the generated code into a new board file inside boards/, for example:
   boards/my_custom_board.py

5) Run the game from the project root:
   python main.py

The game auto-discovers board modules from boards/, so your new map appears without extra config.
